# 森破信使 
让QQ群与MC服务器之间消息互通。 

### 介绍 
这个项目包含**酷Q插件**、**MC服务器插件**和**MC Forge MOD**。 

### 编程语言 
* 酷Q插件：易语言 
* MC服务端插件：Java 
* MC Forge MOD: Java
