# 森破信使 
[![poweredby](https://s.r-ay.cn/svg.php?color=FF69B4&title=提供支持&content=幻创Indev)](https://pacin.dev "前往幻创Indev官网")  

让QQ群和MC服务器或玩家的客户端之间消息互通。**支持原版服务端**！  

### 介绍 
这个项目包含**酷Q插件**、**MC服务端插件**和**配合MC原版服务端的独立软件**。 

### 编程语言 
* 酷Q插件：易语言 
* MC服务端插件：Java 
* 配合原版服务端的森破信使: C#
